# Avito test task

### Для запуска необходимо:
1. Установить Docker compose
2. Установить Postman
3. Создать .env файл и заполнить параметры доступа к БД
```bash
cp .env.example .env
```
4. Запустить контейнер
```bash
make start_env
```
